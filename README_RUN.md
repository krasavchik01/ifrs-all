# KZ-InsurePro - Инструкция по запуску

## Быстрый старт

```bash
cd kz_insurepro
python run.py
```

Открыть в браузере: **http://127.0.0.1:5000/**

## Основные URL

| URL | Описание |
|-----|----------|
| `/` | Главный дашборд страховой компании |
| `/select-role` | Выбор роли (АРФР, ФГСВ, СК) |
| `/arfr-dashboard` | Панель регулятора АРФР |
| `/fgsv-dashboard` | Панель гарантийного фонда ФГСВ |
| `/customers` | CRM - Клиенты |
| `/agents` | CRM - Агенты |

## ВАЖНО

- **Главная страница:** `/` - это рабочий дашборд страховой компании
- **НЕ путать с:** `/landing` - это рекламная презентация, не рабочий интерфейс

## Роли в системе

1. **Страховая компания (insurer)** - главная роль, дашборд с метриками МСФО 17
2. **АРФР (arfr)** - регулятор, видит все страховые компании
3. **ФГСВ (fgsv)** - гарантийный фонд

## Переключение роли

Добавить параметр к URL:
- `/select-role?role=insurer` → дашборд СК
- `/select-role?role=arfr` → панель АРФР
- `/select-role?role=fgsv` → панель ФГСВ

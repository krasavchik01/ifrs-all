Страховые резервы

Постановление АРРФР №304 "Об утверждении Правил формирования системы страховых резервов"

РНП
Резерв незаработанной премии

Часть премии, относящаяся к периоду действия договора после отчётной даты

Методы расчёта:
  • Pro-rata temporis - пропорционально сроку
  • 1/365 - подневной расчёт
  • 1/24 - полумесячный (для краткосрочных)

РНП: {{ "{:,.0f}".format(rnpp_demo.reserve_amount) }} ₸

РЗУ
Резерв заявленных, но неурегулированных убытков

Оценка обязательств по уже заявленным страховым случаям

Метод расчёта:
  • Индивидуальная оценка по каждому убытку
  • На основе акта осмотра / экспертизы
  • С учётом расходов на урегулирование
Компоненты:
РЗУ = Оценка убытка + ALAE (allocated LAE)
- Суброгация - Регресс
РПНУ / IBNR
Резерв произошедших, но незаявленных убытков

Incurred But Not Reported - убытки произошли, но ещё не заявлены

Актуарные методы:
  • Chain-Ladder - коэффициенты развития
  • Bornhuetter-Ferguson - ожидаемый убыток
  • Cape Cod - взвешенный метод
  • Loss Ratio - процент от премии
Требует треугольники развития убытков
РПЗУ
Резерв расходов на урегулирование убытков

Unallocated LAE - общие расходы на урегулирование

Расчёт:
  • % от (РЗУ + РПНУ)
  • Обычно 3-5% для non-life
  • На основе исторических данных
Формула:
РПЗУ = (РЗУ + РПНУ) × k%
Стаб. резерв
Стабилизационный резерв

Для покрытия катастрофических убытков и колебаний убыточности

Применяется для:
  • Катастрофические риски
  • Ядерные риски
  • Риски терроризма
  • Классы с высокой волатильностью
Формируется из прибыли при убыточности ниже нормы
МР (Life)
Математический резерв

Для договоров страхования жизни

Методы расчёта:
  • Prospective - PV будущих выплат - PV премий
  • Retrospective - накопленные премии + инвестдоход - выплаты
Использует таблицы смертности и гарантированную ставку

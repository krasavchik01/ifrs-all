Минимальная Маржа Платежеспособности (ММП):
ММП = max(ММПпремии, ММПубытки) + КорректировкаОГПО ВТС

ММПпремии = Премии × k × 0.18
       = ₸ 80 млрд × 0.70 × 0.18
       = ₸ 10.08 млрд

ММПубытки = Понесенные убытки × k × 0.26
        = ₸ 52 млрд × 0.70 × 0.26
        = ₸ 9.46 млрд

где: k = коэффициент удержания (retention ratio) = 0.70
     КорректировкаОГПО ВТС = ₸ 2.72 млрд (для компаний с ОГПО ВТС)

SCR (Solvency Capital Requirement) по стандартной формуле:
SCR = √(BSCR² + Adj²) + Op

МСФО 9 - Расчет ожидаемых кредитных убытков (ECL)
Постановление АРФР №269 от 29.12.2017, IFRS 9 (п. 5.5)

Общая формула ECL (3-стадийная модель):
ECL = Σt=1T [PDt × LGDt × EADt × DFt]

где:
  PDt = Probability of Default в период t (вероятность дефолта)
  LGDt = Loss Given Default (потери при дефолте)
  EADt = Exposure at Default (подверженность дефолту)
  DFt = Discount Factor = e-EIR×t (дисконтный фактор)
  T = Срок жизни актива (Stage 1: 12 мес, Stage 2/3: до погашения)

Определение стадии (SICR - Significant Increase in Credit Risk):
• Stage 1: PDcurrent / PDorigination ≤ 2.0 → 12-месячный ECL
• Stage 2: PDcurrent / PDorigination > 2.0 или DPD 30-89 дней → пожизненный ECL
• Stage 3: Дефолт или DPD ≥ 90 дней → пожизненный ECL с учетом дефолта

Макроэкономическая корректировка (forward-looking):
PDadjusted = PDbase × [w₁×Mbase + w₂×Madverse + w₃×Msevere]
               = PDbase × [0.55×1.35 + 0.35×1.80 + 0.10×2.40]
               = PDbase × 1.613

Пример: FI-2024-001 (Государственные облигации РК)
Исходные данные:
  EAD0 = ₸ 15,240,000,000 (Gross Carrying Amount)
  PDorigination = 0.01% (т.к. суверенные облигации)
  PDcurrent = 0.02% → Увеличение: 2x
  LGD = 45% (учет восстановления через гарантии)
  EIR = 18.0% (Базовая ставка НБК на дек 2025)
  Срок: 12 месяцев (Stage 1)

Расчет:
  PDadj = 0.0002 × 1.613 = 0.0003226
  DF12m = e-0.18×1 = 0.8353
  EAD12m = ₸ 15,240,000,000 × 0.8353 = ₸ 12,730,072,000

  ECL12m = ₸ 12,730,072,000 × 0.0003226 × 0.45
          = ₸ 1,847,530

МСФО 17 - Оценка обязательств по страховым договорам
IFRS 17 (п. 29-36), Постановление АРФР №376 от 31.12.2019

Общая формула GMM (General Measurement Model):
LICt = BELt + RAt + CSMt

где:
  LIC = Liability for Insurance Contracts (обязательство по страховым договорам)
  BEL = Best Estimate Liability (оценка по наилучшей оценке)
  RA = Risk Adjustment (корректировка на риск)
  CSM = Contractual Service Margin (договорная маржа обслуживания)

1. Расчет BEL (Best Estimate of Liabilities):
BEL = Σt=1n [(CFoutflow,t - CFinflow,t) × DFt]

CFoutflow = Выплатыпо убыткам + Расходыведение дела + Комиссияагентам
CFinflow = Премиибудущие + Возвратперестрахование

DFt = 1 / (1 + rt)t
    где rt = Безрисковая ставка (Kazakhstan Gov Bonds) + Illiquidity Premium
          = 14.5% + 1.2% = 15.7% (на декабрь 2025)

2. Расчет RA (Risk Adjustment) методом CoC:
RA = CoC × Σt=1n [PV(SCRt)]

где:
  CoC = Cost of Capital = 6.0% (IFRS 17.B87, стандартная ставка АРФР)
  SCRt = Solvency Capital Requirement в периоде t
  PV(SCRt) = SCRt × DFt

Альтернативно (CTE метод):
RA = CTE75% - BEL
    где CTE75% = Conditional Tail Expectation на 75-м перцентиле

3. Расчет CSM (Contractual Service Margin):
CSM0 = max(-(BEL0 + RA0 - CF0), 0)

Если CSM0 < 0 → Onerous contract (убыточный договор)
    → Убыток признается немедленно в P&L

Roll-forward CSM (IFRS 17.44):
CSMt+1 = CSMt
          + Interest accretion (CSMt × rlock-in)
          + Changes in fulfillment CF (future service)
          - CSM recognized in P&L (coverage units released)
          ± Foreign exchange impact

Выбор модели оценки (IFRS 17.53-59):
• GMM (General): Долгосрочное страхование жизни с дискреционным участием
• VFA (Variable Fee Approach): Договоры с участием в прибыли (unit-linked)
• PAA (Premium Allocation Approach): Краткосрочные (<12 мес) или immaterial

Пример: Портфель Life Insurance (GMM модель)
Исходные данные:
  Количество договоров: 1,847 (долгосрочное страхование жизни)
  Средний срок: 18.5 лет
  Безрисковая ставка: r = 15.7% (казахстанские суверенные облигации)
  CoC для RA: 6.0%

Расчет BEL (дисконтирование 10 лет):
  PV(Claims) = Σ₁₀ (₸ 7.2 млрд / (1.157)t) = ₸ 36.4 млрд
  PV(Expenses) = Σ₁₀ (₸ 3.8 млрд / (1.157)t) = ₸ 19.2 млрд
  PV(Future Premiums) = Σ₁₀ (₸ 2.1 млрд / (1.157)t) = ₸ 10.1 млрд
  BEL = ₸ 36.4 + ₸ 19.2 - ₸ 10.1 = ₸ 65.5 млрд

Расчет RA (Cost of Capital метод):
  PV(SCR) = Σ₁₀ (₸ 11.4 млрд / (1.157)t) = ₸ 66.7 млрд
  RA = 0.06 × ₸ 66.7 млрд = ₸ 4.0 млрд

Расчет CSM (на момент признания):
  Initial premium inflow = ₸ 90.0 млрд
  BEL0 + RA0 = ₸ 65.5 + ₸ 4.0 = ₸ 69.5 млрд
  CSM0 = ₸ 90.0 - ₸ 69.5 = ₸ 20.5 млрд ✓ Profitable

Итоговое обязательство (LIC):
LIC = BEL + RA + CSM
    = ₸ 65.5 млрд + ₸ 4.0 млрд + ₸ 20.5 млрд
    = ₸ 90.0 млрд
